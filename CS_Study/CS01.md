# CS 공부 

## 1. 컴퓨터 구조 (07-13)

### 1-1. 컴퓨터 구조의 큰 그림
- 컴퓨터가 이해하는 정보 
	+ 데이터
	+ 명령어 : 동작과 수행 대상으로 구성 

- 핵심 부품
	+ CPU
	+ 메모리와 캐시메모리
	+ 보조기억장치
	+ 입출력 장치

### 1. CPU : 정보를 읽고, 해석하고, 실행한다.
- 구성
	+ 산술놀리연산장치(ALU): 연산 회로 집합
	+ 제어장치: 명령어 해석 + 제어신호(전기신호)
	+ 레지스터: CPU내부 중간값 임시 저장 장치

### 2. 메모리 : 실행 중인 프로그램의 데이터와 명령어를 저장한다.
- 특징
	- 주소 : 정보 저장 위치에 접근하기 위한 정보
	- 휘발성 : 전원이 공급되지 않을 때 정보가 지워진다.

- RAM, 메인메모리, 주기억장치로 불린다.

#### 캐시메모리 : CPU의 접근을 위한 중간 저장장치.

### 3. 보조기억장치 : 보관할 대상을 저장한다.
- 특징
	- 비화발성 
	- 안정적, 안전한 보관이 목적
- CD-ROM, DVD, SSD, 하드디스크 등

### 4. 입출력장치 :  컴퓨터 내부와 외부사이 정보를 교환한다
- 보조기억장치는 메모리 보조의 특별 입출력장치이다.

### 5. 부품 간 연결
- 메인보드(마더보드) + 버스(정보를 주고받는 통로)
	- +시스템 버스 - 핵심 부품 연결

### 6. 저장장치 계층
1. 레지스터
2. 캐시메모리
3. 메모리
4. 보조기억장치
특징: 숫자가 작을수록 더 빠르고 더 작고 더 비싸다.
